Anime TV - TV Pro Remote v3

Bu sürüm Android TV kumandası için iki kontrol modu içerir.

1) TV NAV modu (varsayılan)
- Yön tuşları: karttan karta / linkten linke seçim yapar.
- OK: seçili animeyi, bölümü veya butonu açar.
- Seçili öğe mavi çerçeveyle görünür.
- Sayfa sonuna gelince otomatik kaydırır.

2) İmleç modu
- 0 tuşu: TV NAV / imleç modu arasında geçiş yapar.
- 1 tuşu: TV NAV moduna döner.
- 2 tuşu: imleç modunu açar.
- Yön tuşları: beyaz imleci hareket ettirir.
- OK: imlecin olduğu yere dokunur.

Diğer tuşlar
- Menü: ana sayfaya döner.
- Geri: önce sayfada geri gider; imleç modundaysa TV NAV moduna döner.
- Info/Help: kısa kontrol bilgisini gösterir.
- Arama/Red tuşu: sitedeki arama kutusunu seçmeye çalışır.

APK üretimi
GitHub Actions kullanıyorsan Actions > Build Android TV APK > Run workflow yolunu izle.
Artifact: AnimeTV-debug-apk
Kurulacak dosya: app-debug.apk
